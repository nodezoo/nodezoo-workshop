NODEZOO

This is a search engine for Node.js modules.

The results are listed in order of reliability and community support. Each module has a NodeRank:

The more wedges, the better!

Search results give you a summary of the module:

module_name 1.2.3
maintainer's npm username

Short description of the module.

# github 

The code for this engine, including the ranking algorithm, is all on the nodezoo github repository.

Please help the community by suggesting improvements! If you see dodgy results, open a github issue. This will help with tuning.
